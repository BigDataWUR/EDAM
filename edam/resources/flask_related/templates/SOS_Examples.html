SOS server implementation

GetCapabilities

  • Purpose: Returns a metadata for this service
  • Parameters: request = GetCapabilities
  • Example: {{ request.url_root }}SOS/?request=GetCapabilities

DescribeSensor

  • Purpose: Return sensor characteristics
  • Parameters: request = DescribeSensor
    procedure = URN of sensor (see getCapabilities output for sensor's URN)
  • Example: {{ request.url_root }}SOS/?request=DescribeSensor&procedure=2:13:TX

GetObservation

  • Purpose Returns observation data for a station within a user-defined time frame
  • Parameters: request = GetObservation
  • offering = URN of sensor (see getCapabilities output for sensor's URN)
  • observedProperty = (One of : Temperature, Pressure, Nitrogen_Dioxide, Carbon_Monoxide)
  • eventTime = (YYYY-MM-DD HH:MM:SS/YYYY-MM-DD HH:MM:SS). When eventTime = Datetime/ AiRCHIVE returns
    measurements from Datetime until now.
  • Example: {{ request.url_root }}SOS/?request=GetObservation&procedure=2:13:TX
